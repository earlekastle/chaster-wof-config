# chaster-wof-config
Adds import/export buttons to the Wheel of Fortune modal on chaster.app. Allows you to back up a current config to a .JSON file, and import any .JSON configs.

### Installation

1. Install a userscript manager browser extension, like [Tampermonkey](https://www.tampermonkey.net/) (works in Firefox, Chrome, Edge, Safari, Opera)

2. Install the script by [opening the script.user.js file from this repository.](https://github.com/earlekastle/chaster-wof-config/raw/refs/heads/main/script.user.js)
